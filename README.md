# YtScript

** Under Construction **

The cross platform toy scripting language for game development.

## Require
- OCaml
  - Batteries
  - Menhir

- C++
  - Boost
  - SDL2

## Example
```
omake && frontend/src/ytsc input.yts && od -tx1z out.ytx && runtime/ytsguirt/yts-gui-rt
```
